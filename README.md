# Fast Break to Las Vegas: NBA Relocation Toolkit

This project uses data analytics and interactive dashboards to explore the feasibility of relocating an NBA franchise to Las Vegas. It includes:

- custom Streamlit dashboard
- short elevator pitch video
- insights from fan surveys and sports market research
